# Beyond ChatGPT (Draft)

Draft timeline of OpenAI's evolution after the ChatGPT launch. To be researched and refined.

## 2022

- Nov 30: ChatGPT launches as free research preview. 1M users in 5 days.

## 2023

### Monetization & GPT-4 (Jan–Mar)

- Jan 23: Microsoft announces ~$10B investment
- Feb 1: ChatGPT Plus ($20/month)
- Mar 1: gpt-3.5-turbo API launched at $0.002/1K tokens. Whisper API.
- Mar 14: GPT-4 launches (ChatGPT Plus and API)
- Mar 23: ChatGPT plugins announced (alpha): browsing, Code Interpreter, third-party

### Platform Expansion (Jun–Nov)

- Jun 13: Function calling in the API. GPT-3.5 Turbo 16K. Price cuts.
- Jul 6: GPT-4 API generally available. Code Interpreter rolling out.
- Aug 22: GPT-3.5 Turbo fine-tuning
- Aug 28: ChatGPT Enterprise
- Sep 20: DALL-E 3 announced with ChatGPT integration
- Sep 25: ChatGPT voice and image capabilities (GPT-4V)
- Nov 6 (DevDay): GPT-4 Turbo (128K context), GPTs, Assistants API, GPT-4 Vision API, DALL-E 3 API, TTS API

### The Board Crisis (Nov)

- Nov 17: Sam Altman fired. Mira Murati interim CEO. Brockman resigns.
- Nov 19: Emmett Shear named interim CEO
- Nov 20: 700+ employees sign letter threatening to resign
- Nov 21: Altman returns. New board: Bret Taylor (chair), Larry Summers, Adam D'Angelo

### Year-End

- Dec 18: Preparedness Framework (Beta)
- Dec 27: NYT sues OpenAI and Microsoft
- ~100M weekly active users, ~$2B ARR

## 2024

### Q1

- Jan 10: GPT Store launches. ChatGPT Team tier ($25/user/month)
- Jan 25: New embedding models (text-embedding-3-small/large)
- Feb 13: ChatGPT memory feature announced
- Feb 15: Sora announced (research preview, not public)
- Feb 29: Elon Musk sues OpenAI

### Q2

- Apr 9: GPT-4 Turbo with Vision GA. Plugins deprecated.
- Apr 16: Batch API (50% cost discount for async)
- Apr 17: Assistants API v2
- May 8: Model Spec v1 published
- May 13: GPT-4o ("omni") launched — native multimodal, 2x faster, 50% cheaper. ChatGPT macOS app. Scarlett Johansson "Sky" voice controversy.
- May 14: Ilya Sutskever departs
- May 15: Jan Leike departs, criticizes safety culture
- ~May 17: Superalignment team dissolved
- May 28: New Safety and Security Committee
- May 30: ChatGPT Edu
- Jun: Sutskever announces Safe Superintelligence Inc. (SSI)

### Q3

- Jul 18: GPT-4o mini released — replaces GPT-3.5 Turbo. $0.15/M input.
- Jul 25: SearchGPT prototype (limited access)
- Aug: GPT-4o fine-tuning. ~200M weekly active users. ~$3.6B ARR.
- Sep 12: o1-preview and o1-mini — first reasoning models
- Sep 25: Mira Murati, Bob McGrew, Barret Zoph all depart. Advanced Voice Mode rolling out.

### Q4

- Oct 1 (DevDay): Realtime API, vision fine-tuning, model distillation, prompt caching
- Oct 2: $6.6B funding round at $157B valuation
- Oct 3: Canvas launched
- Oct 31: ChatGPT Search launches
- Nov: ChatGPT Windows app. ~300M weekly active users by December.
- Dec 5–20 ("12 Days of OpenAI"): Full o1, ChatGPT Pro ($200/month), Sora public launch, Canvas GA, Apple Intelligence integration, Advanced Voice with video, Projects feature, o3/o3-mini announced
- ~$6B ARR

## 2025

### Q1

- Jan 21: Stargate Project announced ($500B AI infrastructure JV with SoftBank, Oracle, MGX)
- Jan 23: Operator launched (agentic web browsing, Pro users)
- Jan 31: o3-mini released
- Feb 2: Deep Research launched (o3-powered)
- Feb 12: Model Spec v2
- Feb 27: GPT-4.5 released (research preview)

### Q2

- Mar 11: Responses API launched (successor to Chat Completions). Agents SDK (open source).
- Mar 25: GPT-4o native image generation — goes viral (Studio Ghibli trend, 700M+ images, "our GPUs are melting")
- Mar 31: $40B funding round at $300B valuation (SoftBank-led, largest private round ever)
- Apr 14: GPT-4.1, GPT-4.1 mini, GPT-4.1 nano (API, 1M context)
- Apr 15: Preparedness Framework v2
- Apr 16: o3 and o4-mini released. Codex CLI (open source).
- Apr 30: GPT-4 retired from ChatGPT
- May 5–6: Restructuring plan — nonprofit retains control, for-profit becomes PBC, Altman gets no equity
- May 16: Codex cloud agent launched

### Q3–Q4

- Jul: $1B monthly revenue, $12B ARR
- Aug 7: GPT-5 launched — smart routing between fast and thinking models
- Sep 30: Sora 2 launched
- Oct: $6.6B secondary at $500B valuation
- Oct 28: Restructuring completed — OpenAI Foundation (26%), Microsoft (27%), employees/others (47%)
- Nov 12: GPT-5.1
- Dec 11: GPT-5.2
- ~$20B ARR, ~900M weekly active users

## 2026 (through Feb)

- Feb 2: Codex macOS app
- Feb 5: GPT-5.3-Codex
- Feb 13: GPT-4o, GPT-4.1, o4-mini retired from ChatGPT
